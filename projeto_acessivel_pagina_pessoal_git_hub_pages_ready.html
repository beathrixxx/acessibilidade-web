Página Acessível — Beathris

Sobre Contato To‑Do

Oi, eu sou a Beathris

Esta é uma página de demonstração criada para a tarefa de Acessibilidade Web. Está pronta para enviar ao GitHub Pages. Use tab/shift+tab para navegar e experimente um leitor de tela.

O que foi implementado

  • HTML semântico e landmarks
  • Contraste, foco visível e navegação por teclado
  • Labels, atributos ARIA e `alt` nas imagens
  • Formulário com mensagens de erro acessíveis
  • Mini To‑Do que persiste usando localStorage

Formulário de contato

Preencha e clique em Enviar — as mensagens não são realmente enviadas; só demonstram validações acessíveis.

Digite seu nome completo. Campo obrigatório.
Digite um endereço de email válido. Campo obrigatório.
Opcional: escreva uma pequena mensagem.

Mini To‑Do (acessível)

Adicione tarefas, marque como concluída. Funciona com teclado e salva automaticamente.

Digite o texto da tarefa e pressione Enter para adicionar.

Dica: use Tab para navegar e Enter para ativar botões.

Feito com ❤️ para a tarefa de Acessibilidade — pronto para GitHub Pages.
